Chapter 06

GPS Geofencing — Core Feature

Haversine formula tính khoảng cách client-side (R=6,371,000m). Khi distance ≤ Radius → tự động phát TTS → ghi History. Loop 3 giây, không cần server roundtrip.

① Load POI

Online: GET /api/poi → cache SQLite. Offline: GetCachedPOIsAsync(). Lần đầu: seed từ seed_pois.json.

② GPS Tracking

LocationService.GetCurrentLocationAsync(GeolocationAccuracy.Best, 5s)

③ Haversine Check

DistanceHelper.GetDistance(lat1,lon1,lat2,lon2) → mét. Hiển thị "350m" hoặc "1.2km".

④ TTS Playback

GetScriptForLang → TranslateService nếu cần → SpeakAsync + waveform. OnTtsFinished → ghi History.

⑤ Ghi History

Online: POST /api/history — API kiểm tra Role=CUSTOMER. Offline: SQLite PendingHistory → sync khi online.
