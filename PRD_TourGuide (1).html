Chapter 01

Phân Quyền & Vai Trò (RBAC)

3 role với quyền hạn rõ ràng. CUSTOMER bị chặn tại Web CMS — chỉ dùng Mobile App. Owner chỉ thấy POI của mình (filter OwnerId từ Session). Admin thấy tất cả.

🔴
ADMIN
  • Xem & quản lý tất cả POI/Audio/Users
  • Phê duyệt / từ chối POI (PENDING→APPROVED/REJECTED)
  • Xóa hàng loạt POI bị từ chối
  • Xem toàn bộ lịch sử phát + nhật ký hoạt động
  • Dashboard thống kê toàn hệ thống
  • Badge pending POI trên sidebar
🟣
OWNER
  • Chỉ thấy POI có OwnerId = mình
  • Tạo POI → Status=PENDING, chờ Admin duyệt
  • Sửa POI APPROVED → Status=PENDING lại
  • CRUD Audio POI mình (chỉ khi POI APPROVED)
  • Upload ảnh POI tối đa 5 ảnh/5MB
  • Đăng ký /Register → vào luôn, role=OWNER
🟢
CUSTOMER
  • Chỉ dùng Mobile App MAUI Android
  • Bị chặn đăng nhập Web CMS
  • GPS geofencing tự động phát TTS
  • Nghe 4 ngôn ngữ vi/en/ja/zh
  • Offline mode — SQLite cache + seed JSON
  • Lịch sử kiểm soát bởi DB Trigger
POI Approval Workflow: Owner tạo → PENDING (ẩn trên App) → Admin duyệt → APPROVED (hiện trên App). Từ chối → REJECTED kèm lý do. Rút duyệt APPROVED → REJECTED. Owner sửa APPROVED → về PENDING lại. X-Role header phân biệt Admin/Owner tại API.

Chapter 04

Text-to-Speech Pipeline

TTS hoàn toàn miễn phí, không cần API key. Web CMS dùng Web Speech API (browser native). Mobile App dùng MAUI TextToSpeech.SpeakAsync với LocalePriority fallback chain.

📝 Script (vi)→ 🌐 TranslateService (sl=vi)→ 📄 Translated text→ 🎤 ResolveLocaleAsync→ 🔊 SpeakAsync()

📱 Mobile: MAUI TextToSpeech

LocalePriority: vi→[Google Vietnamese, HoaiMy, vi-VN], en→[Google US English, Aria, en-US], ja→[Google 日本語, Nanami, ja-JP], zh→[Google 普通话, Xiaoxiao, zh-CN]. Hỗ trợ Pause/Resume native Android. Waveform 7-bar animation khi phát.

🌐 Web: Browser Speech API

AudioManager preview TTS trực tiếp qua window.speechSynthesis. Dropdown chọn ngôn ngữ nguồn dịch (sl=auto). Auto-translate sang 3 ngôn ngữ còn lại. Google Translate free endpoint không cần key.

Chapter 05

Authentication

🌐 Web CMS — Session Based

HttpContext.Session lưu UserId, Username, Role. CUSTOMER bị chặn với thông báo rõ ràng. Register → role=OWNER, đăng nhập luôn. Auth.cshtml so sánh username/password trực tiếp (chưa hash).

📱 Mobile App — Preferences

AuthService lưu token (GUID random), user_id, name, email, phone vào Preferences. Login chỉ cho CUSTOMER. Guest mode: LoginOfflineAsGuest() với UserId=0 — không ghi được History. ConnectivityService kiểm tra server trước khi login.

Lưu ý bảo mật: PasswordHash trong DB hiện lưu plaintext (chưa bcrypt). Web CMS so sánh trực tiếp. Đây là điểm cần cải thiện cho production — phạm vi đồ án chấp nhận được.

Chapter 06

GPS Geofencing — Core Feature

Tính năng cốt lõi. Haversine formula tính khoảng cách client-side (R=6,371,000m) — không cần server roundtrip. Khi khách vào bán kính POI → tự động phát TTS → ghi History.

Bước 1 — Load POI

Online: GET /api/poi → POIDto[] → LocalDbService.SavePOIsAsync() cache SQLite. Offline: GetCachedPOIsAsync(). Lần đầu chưa có cache: seed từ seed_pois.json (Resources/Raw).

Bước 2 — GPS Tracking

LocationService.GetCurrentLocationAsync() → GeolocationAccuracy.Best, timeout 5s. Fallback: GetLastKnownLocationAsync().

Bước 3 — Haversine Check

DistanceHelper.GetDistance(lat1,lon1,lat2,lon2) → mét. Nếu distance ≤ POI.Radius → trigger. Display DistanceText dạng "350m" hoặc "1.2km".

Bước 4 — TTS Playback

Chọn script theo SettingService.Instance.Language. Nếu lang≠vi → TranslateService.TranslateAsync(). Phát qua TextToSpeechService.SpeakAsync(). Waveform 7-bar animation. OnTtsFinished callback ghi History.

Bước 5 — Ghi History + Offline Queue

Online: POST /api/history {poiId, userId} → API kiểm tra Role + DB Trigger verify CUSTOMER. Offline: AddPendingHistoryAsync() → sync bằng SyncPendingHistoriesAsync() khi có mạng lại.

🗺️ OSRM Routing

MapPage vẽ lộ trình thực tế qua router.project-osrm.org/route/v1/driving. OSRM trả GeoJSON [lon,lat] → đảo thành [lat,lon] → polyline xanh Leaflet. Tự fitBounds sau khi vẽ.

📍 Leaflet via WebView

MapService.BuildLeafletHtml() tạo HTML+JS inline. C#↔JS bridge qua EvaluateJavaScriptAsync(). Bắt URL scheme tourguide://poi/{id} khi tap marker. POI marker highlight/unhighlight qua highlightPOI(id).
