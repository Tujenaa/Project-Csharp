Chapter 01

Phân Quyền & Vai Trò (RBAC)

3 role với quyền hạn rõ ràng. Role CUSTOMER bị chặn hoàn toàn tại Web CMS — chỉ dùng Mobile App. Owner chỉ thấy dữ liệu của mình (filter OwnerId từ Session). Admin thấy tất cả.

🔴
ADMIN
  • Xem & quản lý TẤT CẢ POI
  • Xem & quản lý TẤT CẢ Audio
  • Quản lý Users (CRUD + đổi role)
  • Phê duyệt / từ chối POI (PENDING → APPROVED/REJECTED)
  • Xóa hàng loạt POI bị từ chối
  • Xem toàn bộ lịch sử phát
  • Dashboard thống kê toàn hệ thống
  • Menu Phê duyệt POI chỉ hiện cho ADMIN
🟣
OWNER
  • Chỉ thấy POI có OwnerId = mình
  • Tạo POI → status PENDING, chờ Admin duyệt
  • Sửa POI đang APPROVED → status PENDING lại
  • CRUD Audio của POI mình sở hữu
  • Xem lịch sử phát POI của mình
  • Upload ảnh POI (lưu vào bảng POIImages)
  • Dashboard thống kê riêng
  • Sửa thông tin cá nhân (/Profile)
🟢
CUSTOMER
  • Chỉ dùng Mobile App MAUI Android
  • Bị chặn đăng nhập Web CMS
  • GPS geofencing tự động phát TTS
  • Nghe TTS 4 ngôn ngữ vi/en/ja/zh
  • Offline mode: đọc POI từ SQLite cache
  • Lịch sử phát kiểm soát bởi DB Trigger
  • Bản đồ Leaflet real-time + vẽ lộ trình OSRM
  • Đăng ký → tạo account CUSTOMER
POI Approval Workflow: Owner tạo POI → Status = PENDING → POI không hiện trong App. Admin vào trang Phê duyệt → Duyệt → Status = APPROVED → hiện trong App. Admin có thể Từ chối (Status = REJECTED) kèm lý do, hoặc Rút duyệt APPROVED → về REJECTED. Xóa tất cả REJECTED bằng một nút.

Chapter 04

Text-to-Speech Pipeline

TTS hoàn toàn miễn phí, không cần API key. 4 ngôn ngữ vi/en/ja/zh. Web CMS dùng Web Speech API (browser native). Mobile App dùng MAUI TextToSpeech.SpeakAsync + fallback chain chọn giọng tốt nhất.

Fallback Chain theo ngôn ngữ (LocalePriority)

🇻🇳 vi
Tiếng Việt (Script gốc — không dịch) Ưu tiên: Google Vietnamese → Microsoft HoaiMy → vi-VN. Script gốc luôn là tiếng Việt, không qua Google Translate.
🇺🇸 en
Tiếng Anh Ưu tiên: Google US English → Microsoft Aria → en-US. Script dịch từ vi → en qua Google Translate free endpoint.
🇯🇵 ja
Tiếng Nhật Ưu tiên: Google 日本語 → Microsoft Nanami → ja-JP. TranslateService dịch vi → ja trước khi phát.
🇨🇳 zh
Tiếng Trung (Phổ thông) Ưu tiên: Google 普通话 → Microsoft Xiaoxiao → zh-CN. Google Translate free tier (không cần key).
TranslateService (Mobile App): Gọi https://translate.googleapis.com/translate_a/single?client=gtx&sl=vi&tl={lang}&dt=t&q={text}. Có retry wrapper (maxRetries=2). Nếu dịch thất bại → fallback phát tiếng Việt gốc. User-Agent bắt buộc để Google không trả 403.

📱 Mobile: Waveform Animation

PlaceDetailPage có 7 bar BoxView animate khi TTS đang phát. Mỗi bar có peak height và delay khác nhau tạo hiệu ứng sóng âm. CancellationToken dừng animation khi TTS kết thúc. OnTtsFinished callback ghi History và cập nhật UI.

🌐 Web: Browser Preview

AudioManager cho phép Owner preview TTS trực tiếp trong browser bằng window.speechSynthesis.speak(utterance). Hỗ trợ Play/Pause/Resume/Stop cho từng hàng. Auto-translate bằng Google Translate free khi bấm nút dịch.

Chapter 06

GPS Geofencing — Core Feature

Tính năng cốt lõi. Sử dụng Haversine formula tính khoảng cách client-side, không cần server. Khi khách vào bán kính POI → tự động phát TTS → ghi History. LocationService + DistanceHelper.GetDistance().

Bước 1 — Load POI từ API hoặc SQLite Cache

ApiService.GetPOI() → nếu online: gọi /api/poi lấy POIDto → cache xuống SQLite (LocalDbService.SavePOIsAsync). Nếu offline: đọc từ GetCachedPOIsAsync(). File seed mặc định: seed_pois.json trong Resources/Raw.

Bước 2 — GPS Tracking

LocationService.GetCurrentLocationAsync() → GeolocationAccuracy.Best, timeout 5s. Fallback: GetLastKnownLocationAsync(). Lọc nhiễu bằng accuracy threshold. ConnectivityService.IsConnected kiểm tra mạng trước khi gọi API.

Bước 3 — Haversine Distance Check

DistanceHelper.GetDistance(lat1, lon1, lat2, lon2) → trả về khoảng cách mét. Dùng công thức Haversine trên mặt cầu R=6,371,000m. So sánh với POI.Radius. Display: DistanceText (350m / 1.2km).

Bước 4 — TTS Playback

Chọn script theo SettingService.Instance.Language: vi/en/ja/zh. Nếu lang ≠ vi → TranslateService.TranslateAsync(script, toLang). Phát bằng TextToSpeechService.SpeakAsync() với LocalePriority chain. Waveform animation 7 bars.

Bước 5 — Ghi History + Offline Queue

Online: POST /api/history {poiId, userId} → DB Trigger kiểm tra Role=CUSTOMER. Offline: LocalDbService.AddPendingHistoryAsync(). Khi có mạng lại: SyncPendingHistoriesAsync() sync queue lên server. HistoryStore.AddAsync(poi) cập nhật UI real-time.

🗺️ OSRM Route Drawing

MapPage hỗ trợ vẽ lộ trình thực tế qua https://router.project-osrm.org/route/v1/driving. OSRM trả về GeoJSON coordinates → vẽ polyline màu xanh lên Leaflet map. Tự động fitBounds sau khi vẽ.

📍 Leaflet via WebView

MapService.BuildLeafletHtml() tạo HTML string với Leaflet.js nhúng. C# ↔ JS bridge qua mapWebView.EvaluateJavaScriptAsync(js). Bắt URL scheme tourguide://poi/{id} để C# xử lý khi tap marker.

sequenceDiagram
    participant C as Customer (Phone)
    participant APP as TourGuideApp (MAUI)
    participant GEO as LocationService
    participant DH as DistanceHelper
    participant TTS as TextToSpeechService
    participant TR as TranslateService
    participant API as TourGuideAPI
    participant DB as SQL Server

    APP->>API: GET /api/poi → POIDto[] (APPROVED only)
    API->>DB: JOIN Audio WHERE Status=APPROVED
    DB-->>API: POIDto list
    API-->>APP: POIs với ScriptVi/En/Ja/Zh + Images[]
    APP->>APP: LocalDbService.SavePOIsAsync() (cache SQLite)

    loop GPS Update
        APP->>GEO: GetCurrentLocationAsync(GeolocationAccuracy.Best)
        GEO-->>APP: Location {lat, lng, accuracy}
        
        loop Foreach POI
            APP->>DH: GetDistance(userLat, userLng, poi.Lat, poi.Lng)
            DH-->>APP: distance (metres, Haversine)
            
            alt distance ≤ poi.Radius
                APP->>APP: GetScriptForLang(poi, SettingService.Language)
                
                alt Language == "vi"
                    APP->>TTS: LoadText(poi.ScriptVi)
                else Language != "vi"
                    APP->>TR: TranslateAsync(poi.ScriptVi, lang)
                    TR->>TR: GET translate.googleapis.com (free)
                    TR-->>APP: translatedText
                    APP->>TTS: LoadText(translatedText)
                end
                
                APP->>TTS: SpeakAsync(cancelToken)
                TTS->>TTS: ResolveLocaleAsync(lang, availableLocales)
                TTS-->>C: 🔊 TTS phát + waveform animation

                TTS->>APP: OnFinished event
                APP->>API: POST /api/history {PoiId, UserId}
                API->>DB: INSERT INTO History
                DB->>DB: Trigger: verify Role=CUSTOMER
                DB-->>API: OK
                API-->>APP: History saved
            end
        end
    end
